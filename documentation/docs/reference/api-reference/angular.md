The `@pivothead/angular` package provides a service for creating your **headless pivot table**.

**`PivotTableService`**

- **`createPivotTable(data, config)`**: A method that generates the pivot table.
  - **`data`**: `Array<Object>`
  - **`config`**: `Object`
  - **Returns**: `Object` - The structured pivot table data.
